LumenX Features

Code Analysis Dimensions

  • Coverage Analysis - Unit, integration, E2E tests
  • Quality Metrics - Complexity, duplication, linting
  • Performance Scoring - Latency, Core Web Vitals
  • Security Analysis - Vulnerabilities, insecure patterns
  • SEO Optimization - Meta tags, Open Graph, structured data
  • Documentation Quality - README, API docs, comments
  • UI/UX Review - Layout, responsive, accessibility

lumenx scan --path .
